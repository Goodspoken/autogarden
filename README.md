# Autogarden: Arduino Auto-watering System 🌱

Продвинутая система автоматического полива и контроля микроклимата на базе **Arduino Nano** и **PlatformIO**.

## Описание проекта

Autogarden — это DIY-проект автоматизации ухода за растениями, который берет на себя рутину по поливу и контролю температуры/освещения. Система считывает показания с датчиков влажности почвы и освещенности, управляет водяными насосами и системой вентиляции.

Все управление осуществляется с помощью удобного меню на LCD дисплее (I2C) с использованием энкодера.

## 🛠 Особенности и железо (Hardware)

*   **Микроконтроллер:** Arduino Nano (ATmega328P)
*   **Экран:** LCD 1602 с модулем I2C (PCF8574T)
*   **Время:** Модуль реального времени (RTC) DS3231
*   **Датчики:**
    *   2x Аналоговых датчика влажности почвы
    *   2x Аналоговых датчика света
*   **Управление (Выходы):**
    *   2-х канальное реле (управление насосами 12В)
    *   4-х канальное реле (управление кулерами/вентиляторами по свету)
*   **Ввод (Пользовательский интерфейс):**
    *   Поворотный энкодер с кнопкой для навигации по меню
    *   Физические кнопки включения Помпы 1 и Помпы 2
    *   Кнопка "Домой" для быстрого возврата в главное меню

## 💻 Программная часть (Software)

Проект изначально настроен для работы в среде **PlatformIO**. 

В проекте используются следующие библиотеки (настраиваются автоматически через `platformio.ini`):
*   `LiquidCrystal_PCF8574`
*   `RTClib`
*   *Опционально:* `FastLED` и библиотеки `Adafruit` для экранов.

### Установка и запуск

1. Клонируйте репозиторий.
2. Откройте папку проекта в VS Code (или Antigravity IDE) с установленным расширением **pioarduino** (или PlatformIO).
3. Дождитесь, пока IDE автоматически скачает все зависимости из `platformio.ini`.
4. Скомпилируйте (Build) и загрузите (Upload) код в Arduino Nano.
